# 🥗 NutriScan

AI-powered food calorie and nutrition analyzer for iOS.

## 📱 Features

- Take photos of food using the camera
- Automatic food recognition with Google Gemini AI
- Calorie and nutrition analysis
- Clean and user-friendly interface

## 🔧 Tech Stack

- Swift & SwiftUI
- Google Gemini 2.5 Flash API
- UIKit (Camera integration)
- REST API integration

## 🚀 Setup

1. Clone the repository
2. Get a Gemini API key from [Google AI Studio](https://aistudio.google.com)
3. Replace `YOUR_API_KEY` with your key in `ContentView.swift`
4. Run with Xcode
